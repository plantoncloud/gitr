# gitr

Tool to clone git repositories and locate git repo on SCM using Browser URLs

### Install

```
brew tap swarupdonepudi/homebrew-gitr
brew install gitr
```

### Test

```
clone https://github.com/swarupdonepudi/gitr
```

### Cleanup

```
brew uninstall gitr
brew untap swarupdonepudi/homebrew-gitr
```
